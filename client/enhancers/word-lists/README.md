| Entity      | Description                                         |

| PERCENT     | Percentage, including "%"                            |
| PERSON      | People, including fictional                          |
| PRODUCT     | Objects, vehicles, foods, etc. (not services)        |
| QUANTITY    | Measurements, as of weight or distance               |
| TIME        | Times smaller than a day                             |
